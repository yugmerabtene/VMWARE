# Cours VMware vSphere

> **Dernière mise à jour :** 2026

---

## Présentation

Ce cours couvre l'administration complète de l'infrastructure VMware vSphere, de l'installation d'ESXi jusqu'à la supervision avancée. Il est structuré en 4 niveaux de progression.

---

## Prérequis

- Connaissances de base en administration système (Linux/Windows)
- Notions de réseau (TCP/IP, VLAN, DNS)
- Familiarité avec les concepts de stockage (SAN, NAS)
